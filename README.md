## Supported Formats

**Currently there are only implementations for image formats, but more mediums will be added!**

### Input
- `webp`
- `png`
- `jpg`
- `tiff`
- `gif`
- `bmp`
- `ico`

### Output
- `png`
- `jpg`
- `tiff`
- `gif`
- `bmp`
